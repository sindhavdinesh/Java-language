# 🦠 Corona Tracker – COVID-19 API Project

## 📌 Overview

Corona Tracker is a web-based application that fetches and displays real-time COVID-19 data using a public API.

The application provides updated statistics such as total cases, recoveries, and deaths in a simple and user-friendly interface.

---

## 🚀 Features

* 🌍 Fetches real-time COVID-19 data from API
* 📊 Displays total cases, recovered, and deaths
* ⚡ Fast and dynamic data loading
* 🧑‍💻 Clean and responsive UI
* 🔄 Auto-updated data using API calls

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* API: Public COVID-19 API
